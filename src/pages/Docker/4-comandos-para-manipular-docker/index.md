## Comandos Básicos para manipular Docker
Existen varios comandos, y cad comando es para una función, ya sea eliminar. crear ,etc.
Listamos algunos comandos más utilizados :
```
Comando							Descripción
docker attach	  Adjunte flujos de entrada, salida y error  local a un contenedor en ejecución
docker build	  Contruye una imágen desde un Dockerfile
docker checkpoint Administra checkpoints
docker commit	  Crea una nueva imágen desde los cambios de un contenedor existente
docker config	  Administra las configuraciones de Docker (configs)
docker container  Administra los contenedores
docker cp		  Copia archivos/carpetas entre un contenedor y el sistemas de archivos local
docker create	  Crea un nuevo contenedor
docker deploy	  Implementa(Deploy) una nueva pila o actualiza una pila existente
docker diff		  Inspecciona los cambios a los archivos o directorios en el sistema de archivos de un contenedor
docker events	  Obtiene eventos en tiempo real desde el servidor(server)
docker exec		  Ejecuta un comando en un contenedor en ejecución
docker export	  Exporta un  filesystem de un contenedor a un archivo tar
docker history	  Muestra el historial de una imágen
docker image	  Administra las imágenes
docker images	  Lista la imágenes
docker import	  Importa el contenido de un tarball para crear una imagen
docker info		  Muestra información de todo el sistema
docker inspect	  Retorna información de bajo nivel en objetos Docker
docker kill		  Kill uno o mas contenedores en ejecución
docker load		  Carga una imagen desde un archivo o STDIN
docker login	  Log in to a Docker registry
docker logout	  Salir de un registro(log) de Docker
docker logs		  Obtener los registros(logs) de un contenedor
docker network	  Administra networks (redes)
docker node		  Administra nodos Swarm
docker pause	  Pausa todos los procesos dentro de uno o más contenedores
docker plugin	  Administra plugins
docker port		  Lista mapeos de puertos o un mapeo específico para un contenedor
docker ps		  Lista los contenedores
docker pull		  Pull (descarga) una images o repositorio desde un registro
docker push		  Push (sube) una imagen o repositorio al registro
docker rename	  Cambia el nombre a  un contenedor
docker Restart	  Reinicia uno o más contenedores
docker rm		  Remueve uno o más contenedores
docker rmi		  Elimina una o más imágenes
docker run		  Ejecuta un comando en un contenedor nuevo
docker save		  Guarde una o más imágenes en un archivo tar
docker search	  Busca en el Docker Hub  para obtener imagenes
docker secret	  Administra Docker secrets
docker service	  Administra servicios
docker stack	  Administra las pilas de docker (stacks)
docker start	  Inicia uno o más contenedores detenidos
docker stats	  Muestra en vivo estadísticas de uso de recursos de contenedores.
docker stop		  Detiene uno o más contenedores en ejecución
docker swarm	  Administra Swarm
docker system	  Administrar Docker
docker tag		  Crea un tag TARGET_IMAGE el cual refiere a la  SOURCE_IMAGE
docker top		  Muestra los procesos en ejecución de un contenedor
docker trust	  Administra la confianza en las imágenes de Docker (experimental)
docker unpause	  Unpause  todos los procesos dentro de uno o más contenedores
docker Update	  Actualizar la configuración de uno o más contenedores
docker version	  Mostrar la información de la versión Docker
docker volume	  Administrar volúmenes
docker wait		  Bloquea hasta que se detengan uno o más contenedores e imprime sus códigos de salida
```
